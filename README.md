# Data Science Portfolio

This repository contains my personal Data Science Portfolio, built with React and Vite.

## Features
- Showcases various data science projects.
- Includes sections for Hero, About, Projects, and Education.
- Professional and responsive design.

## Tech Stack
- React.js
- Vite
- Lucide React (Icons)
- Vanilla CSS

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
